Este es un proyecto base para trabajar con la librería wxWidgets. 
Solo define la clase Aplication, pero no crea  ninguna ventana. 

Estan definidas las opciones de compilación y enlazado para los 
componentes más habituales para que pueda compilar su proyecto.
En Windows, estas opciones utilzan la versión 3.x, compilada en 
modo unicode, y por defecto enlazada dinámicamente.

Puede acceder a la documentación de la biblioteca mediante la opción
"Referencia wxWidgets..." del submenu "Diseñar Interfases" del menu 
"Herramientas".
